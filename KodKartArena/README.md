# DevCard Arena 🃏🚀

**DevCard Arena**, yazılımcıların yeteneklerini dijital kartlar halini getirerek sergilediği, rekabet ettiği ve gelişimini takip ettiği bir mobil platformdur.

## 🎯 Projenin Amacı ve Oyunlaştırma Özellikleri

Projenin temel amacı, yazılım gelişim sürecini bir oyun haline getirerek motivasyonu artırmaktır.

### Oyunlaştırma Mekanikleri:
- **XP Sistemi:** Yeni beceriler kazandıkça ve projeler tamamladıkça tecrübe puanı (XP) kazanın.
- **Seviye Atlama:** XP biriktirerek "Junior"dan "Legendary" seviyesine kadar yükselin.
- **Başarı Başarımları (Achievements):** Özel görevleri (Örn: "Clean Coder", "UI Master") tamamlayarak rozetler kazanın.
- **Liderlik Tablosu (Leaderboard):** Global sıralamada diğer geliştiricilerle rekabet edin.
- **Dijital Kart Kartvizit:** Kendi "DevCard"ınızı oluşturun ve diğer geliştiricilerin kartlarını inceleyin.

## ⚙️ Nasıl Çalıştırılır? (Installation & Run)

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

### 1. Gereksinimler
- Node.js (LTS sürümü önerilir)
- Expo Go (Mobil cihazınızda)

### 2. Kurulum
```bash
# Depoyu klonlayın
git clone <repo-url>
cd yazilimci_kimlik_kart

# Bağımlılıkları yükleyin
npm install
```

### 3. Uygulamayı Başlatma
```bash
# Geliştirme sunucusunu çalıştırın
npx expo start
```
Terminalde çıkan **QR kodu**, mobil cihazınızdaki **Expo Go** uygulaması ile taratarak uygulamayı anında görüntüleyebilirsiniz.
