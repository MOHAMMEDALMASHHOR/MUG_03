# KodKart Arena 🃏🚀

Geliştirici Profilini Oyunlaştıran Geleceğin Platformu! KodKart Arena, yazılım becerilerini, deneyimini ve mevcut çalışma durumunu eğlenceli bir oyun mekaniği ile sergileyebileceğin interaktif bir yazılımcı kimlik kartıdır.

---

## 🎯 Projenin Amacı ve Oyunlaştırma Özellikleri

- **XP Sistemi:** Çalıştıkça ve yeni projelere dahil oldukça **XP (Deneyim Puanı)** kazanırsınız. (-20 Enerji harcayarak, +35 XP kazanın!)
- **Seviye Atlama:** Topladığınız XP miktarına göre geliştirici unvanınız dinamik olarak güncellenir:
  - 🥉 **Junior** (0 - 99 XP)
  - 🥈 **Mid-Level** (100 - 249 XP)
  - 🥇 **Senior** (250+ XP)
- **Enerji Barı:** Gerçek bir yazılımcı gibi kod yazarken yorulursunuz. "Projeye Odaklanmış Durumda" enerjiniz düşer.
- **Kahve Mekaniği ☕:** Enerjiniz bittiğinde tek bir "Kahve İç" butonuna tıklayarak +30 Enerji toplayın ve üretkenliğinize geri dönün!
- **Dinamik Geri Bildirim:** Kartın kenarlıkları, animasyonları ve durumu (Yeşil: Müsait, Kırmızı: Çalışıyor) unvanınıza ve iş durumunuza göre anında tepki verir. Dark Mode & Glassmorphism tasarımıyla modern bir UI/UX deneyimi yaşatır.

---

## ⚙️ Nasıl Çalıştırılır? (Installation & Run)

Projeyi kendi ortamında test etmek çok kolay! Aşağıdaki adımları sırasıyla ve eksiksiz bir şekilde uygulayabilirsin.

**Gereksinimler:** Node.js (v18+) ve mobil testler için telefonunda `Expo Go` uygulaması yüklü olmalıdır.

```bash
# Repoyu bilgisayarına yedeğini al/klonla (Veya ZIP Olarak İndir)
git clone https://github.com/MOHAMMEDALMASHHOR/MUG_03.git

# Proje dizinine gir
cd MUG_03/KodKartArena

# Gerekli kütüphaneleri ve bağımlılıkları kur
npm install

# Projeyi başlat
npx expo start
```
*Geliştirici Notu:* Terminalde açılan QR kodu telefonunun kamerasıyla / Expo Go uygulamasıyla taratarak projeyi anında telefonunda canlı test edebilirsin!
