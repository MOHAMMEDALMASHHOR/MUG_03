Build a React Native mobile application called "DevCard Arena" — a gamified developer 
profile card system. Here is the full specification:

---

## CORE CONCEPT

A developer identity card app with deep gamification. Each card represents a 
developer with real-time hire/availability status, XP system, level progression, 
achievements, and animated badge unlocks.

---

## TECH STACK
- React Native (Expo or bare workflow)
- TypeScript
- React Navigation (stack + bottom tabs)
- Reanimated 2 (for all animations)
- AsyncStorage (local persistence)
- react-native-linear-gradient
- @shopify/flash-list (for card lists)
- expo-haptics (haptic feedback on interactions)

## DATA MODEL
```typescript
interface Developer {
  id: string;
  name: string;              // e.g. "Ayşe Kaya"
  title: string;             // e.g. "Senior React Native Dev"
  specialization: string;   // e.g. "Mobile", "Backend", "AI/ML", "DevOps", "Frontend"
  level: DeveloperLevel;    // "Junior" | "Mid" | "Senior" | "Lead" | "Principal" | "Legendary"
  xp: number;               // 0–10000
  xpToNextLevel: number;
  isAvailable: boolean;
  avatar: string;           // emoji or initials-based avatar
  badges: Badge[];
  achievements: Achievement[];
  hireCount: number;        // how many times this card has been "hired"
  endorsements: Endorsement[];
  streak: number;           // consecutive days active
  joinedAt: Date;
  techStack: string[];      // ["React Native", "TypeScript", "GraphQL"]
  rank: number;             // global leaderboard rank
}

interface Badge {
  id: string;
  name: string;
  icon: string;             // emoji
  description: string;
  rarity: "common" | "rare" | "epic" | "legendary";
  unlockedAt?: Date;
  isUnlocked: boolean;
}

interface Achievement {
  id: string;
  title: string;
  description: string;
  icon: string;
  progress: number;         // 0–100
  isCompleted: boolean;
  xpReward: number;
  completedAt?: Date;
}
```

## THE DEV CARD COMPONENT

This is the centerpiece. Design it like a premium trading card:

Visual Layers (back to front):
1. Gradient background based on specialization:
   - Mobile: Deep purple → electric indigo
   - Backend: Dark teal → emerald
   - AI/ML: Midnight blue → neon cyan
   - DevOps: Charcoal → orange flame
   - Frontend: Rose gold → coral
2. Subtle grid/circuit pattern overlay (SVG path)
3. Holographic shimmer effect (Animated linear gradient sweeping diagonally)
4. Card content: avatar, name, title, level badge, XP bar, tech stack
5. Rarity glow border based on level:
   - Junior: gray
   - Mid: green
   - Senior: blue
   - Lead: purple
   - Principal: gold
   - Legendary: rainbow animated border

Card States:
- Available: glowing green dot + "AVAILABLE FOR HIRE" tag
- Unavailable: dimmed overlay + "WORKING ON PROJECTS" tag with busy animation

Card Flip Animation:
- On long press: card flips 180° (Reanimated2 rotateY)
- Back shows: achievements, join date, hire count, streak

## GAMIFICATION SYSTEM

### XP & Leveling
- Level thresholds: Junior(0) → Mid(500) → Senior(1500) → Lead(3000) → Principal(6000) → Legendary(10000)
- XP earned by:
  - Getting hired: +50 XP + haptic pulse
  - Profile view: +5 XP
  - Achievement unlock: variable XP
  - Daily login streak: +10 XP/day, multiplier at 7-day streak
- Level-up: full-screen particle explosion animation + level badge morphs

### Hire Button Behavior
When "Hire" button is pressed:
1. Button scales down (spring) then back up
2. Haptic impact feedback (heavy)
3. XP counter increments with rolling number animation
4. isAvailable flips to false
5. Button text animates from "Hire Developer" → "Working on Projects" with fade
6. A floating "+50 XP" text flies upward and fades out
7. Check if level-up triggered → if yes, show LevelUpModal

### Achievement Examples (build all of these):
- "First Hire" — Get hired for the first time (50 XP)
- "Hot Streak" — Maintain 7-day login streak (100 XP)
- "Tech Polyglot" — Add 5+ technologies to your stack (75 XP)
- "Elite Status" — Reach Senior level (200 XP)
- "Legendary Dev" — Reach Legendary level (500 XP)
- "Popular Pick" — Get hired 10 times (150 XP)
- "Speedrun" — Create a card in under 60 seconds (25 XP)
- "Full Stack" — Have both Frontend and Backend in tech stack (60 XP)

### Badges (unlockable, displayed on card):
- 🔥 "On Fire" — 3 hires in one day
- ⚡ "Speed Coder" — Complete profile in record time
- 💎 "Diamond Dev" — Reach Principal level
- 🌟 "Rising Star" — Top 10 on leaderboard
- 🎯 "Specialist" — 100% profile completeness
- 🏆 "Champion" — Most hired developer of the week

Badge rarity visuals:
- Common: gray border, no glow
- Rare: blue shimmer
- Epic: purple pulse animation
- Legendary: gold + particle orbit animation

## ANIMATIONS (use Reanimated 2 throughout)

1. Card enter animation: slide up + fade in with spring (staggered in list)
2. XP bar fill: spring-based width interpolation on mount
3. Level-up modal: scale from 0 → 1.1 → 1 (elastic), particle system behind
4. Badge unlock: flip card animation, then badge slot glows and badge drops in
5. Hire button press: scale 0.95 → 1.05 → 1 with haptics
6. "+XP" floating text: translateY from 0 → -60, opacity 1 → 0 over 1s
7. Holographic card shimmer: continuous looping gradient sweep
8. Achievement progress ring: SVG stroke-dashoffset animation
9. Leaderboard rank change: color flash + arrow bounce
10. Number counters: rolling/slot-machine style increment animation
